🔧 Usage Examples

Simple HTML usage:

<kids-keyboard learning-mode="associations"></kids-keyboard>

Programmatic usage:

import { createKidsKeyboard } from './src_new/index.js'; const keyboard = createKidsKeyboard('#container', { learningMode: 'lessons', enableAudio: true, autoTutor: true });

Feature access:

import { startLesson, getSessionStats } from './src_new/index.js'; // Start a typing lesson startLesson(container, 'beginner'); // Get usage statistics const stats = getSessionStats();
🎯 How It Works

Following the functional web component pattern:

  1. Pure data parsing - Extract configuration from attributes
  2. Functional HTML generation - Create markup from data
  3. Feature initialization - Set up audio, display, lessons
  4. Event handling - Process user interactions functionally
  5. State management - Immutable state updates

The web component class is just a thin browser interface - all the real logic lives in pure, testable functions.
